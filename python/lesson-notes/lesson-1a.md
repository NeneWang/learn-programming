## Printing numbers
```python
# This program displays some stuff  # Line 3
print ("Welcome to Python Programming")    # Line 4
print ("Try coding, it's fun. Do you know what is 123456789*123456789?") # Line 5
print (123456789*123456789)    # Line 6
```
Output
```output
# This program displays some stuff  # Line 3
print ("Welcome to Python Programming")    # Line 4
```
