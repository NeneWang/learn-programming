## Basic Math Operations using Variables

```java
  int num = 100;
  // Addition
  int sum    = 20 + 10;
  System.out.println(sum);

  // Subtraction
  int sub    = 20 - 10;
  System.out.println(sub);

  // Multiplication
  int mul    = 20 * 10;
  System.out.println(mul);

  // Division
  int div    = 20 - 10;
  System.out.println(div);

```
